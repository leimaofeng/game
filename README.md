# 围棋游戏 (Go/Weiqi Game)

一款支持双人对弈和人机对战的围棋游戏，提供微信小游戏版本和网页版本。

## 功能特性

- 🎮 **19×19 标准棋盘** - 带星位标记的完整围棋棋盘
- 👥 **双人对弈** - 两人轮流落子
- 🤖 **人机对战** - 三种难度（简单/中等/困难），可选执黑或执白
- ↩️ **悔棋功能** - 支持撤销操作
- 💾 **存档系统** - 5个存档位 + 自动保存
- ✅ **完整规则** - 气的计算、提子、打劫禁入、禁止自杀、Pass、中国规则计目

## 项目结构

```
├── game.js                 # 微信小游戏入口
├── game.json               # 微信小游戏配置
├── project.config.json     # 微信小游戏项目配置
├── js/
│   ├── main.js             # 主游戏逻辑
│   └── game/
│       ├── rules.js        # 围棋规则引擎
│       ├── board.js        # 棋盘渲染
│       ├── ai.js           # AI对战模块
│       ├── storage.js      # 存档管理
│       └── ui.js           # UI界面管理
└── web/
    └── index.html          # 网页版（可直接浏览器打开）
```

## 快速开始

### 网页版（推荐体验）

直接用浏览器打开 `web/index.html` 即可游玩。

### 微信小游戏版

1. 下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 打开微信开发者工具，选择「导入项目」
3. 选择项目根目录，AppID 可使用测试号
4. 点击编译即可在模拟器中运行

## 游戏操作

| 操作 | 说明 |
|------|------|
| 点击棋盘 | 在交叉点落子 |
| 悔棋 | 撤销上一步（人机模式撤销两步） |
| 停一手 | Pass，双方连续停则终局计分 |
| 保存 | 保存当前棋局到存档位 |
| 认输 | 直接结束对局 |

## 技术实现

- **围棋规则引擎**：完整实现气的计算、提子判定、打劫检测、禁止自杀等规则
- **AI算法**：基于评估函数的AI，考虑位置价值、连接、威胁、防守等因素
- **渲染**：Canvas 2D 绘制，木纹棋盘 + 光泽棋子效果
- **存储**：localStorage（网页版）/ wx.setStorage（小游戏版）
